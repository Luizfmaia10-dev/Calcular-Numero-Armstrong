# Calcular-Numero-Armstrong
Um Número Armstrong (também conhecido como número narcisista) é um conceito matemático curioso onde um número é igual à soma de seus próprios dígitos, cada um elevado à potência do número de dígitos do número total.

Em termos simples: se um número tem $n$ dígitos, você eleva cada dígito à potência $n$ e soma tudo. Se o resultado for o número original, ele é um Número Armstrong.

Na minha primeira prova de AEDS II(Algoritmos e estruturas de dados) o meu professor cobrou essas questão mas de uma maneira recursiva,o que me deixou bem confuso,mas depois dele explicar direito o que cada parte faz eu compreendi melhor o codigo e agora estou aqui mostrando para voces a maneira certa de realizar essa questão

Neste exercício, foquei em aplicar pilares fundamentais da ciência da computação e do desenvolvimento Java:

Recursividade: A lógica principal de soma das potências foi implementada de forma recursiva, demonstrando o domínio sobre casos base e empilhamento de chamadas (call stack).

Manipulação de Tipos e Casting: Uso da classe Math.pow (que trabalha com double) e a conversão precisa para int, garantindo a integridade dos cálculos matemáticos.

Entrada de Dados (I/O): Implementação da classe Scanner para interação em tempo real com o usuário via console.

Lógica Matemática: Criação de algoritmos para contagem de dígitos e extração de algarismos usando operadores de módulo (%) e divisão inteira (/).

Clean Code: Organização de métodos com responsabilidades únicas (cont_digitos, calculaArmstrong, numeroArmstrong), facilitando a leitura e manutenção do código.
